How to read position and button states from the Novint Falcon:

Run the following node:
	ros2 run falcon_ros2 falcon_node

This node publishes to:
	/falcon/button_raw
	/falcon/position

This node subscribes to:
	/falcon/force
	
Current range of position readings:
	-0.062 < x < 0.062
	-0.058 < y < 0.058
	 0.075 < z < 0.175

Button Bit Mask -> Individual Button State:
	right -> 1
	up -> 2
	middle -> 4
	left -> 8
	
- to check the states of each button, take the raw bit mask and use the AND operator with the button you want
- ex. if you want the state of the middle button: raw & 4 = 1 if pressed and 0 if not pressed
